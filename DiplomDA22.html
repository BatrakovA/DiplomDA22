Введение

         Целью данной работы является анализ набора параметров футболистов игры FIFA 2019.

         FIFA 2019 - 26-я футбольная игра из серии игр FIFA, разработанная для платформ Windows, Nintendo Switch, PlayStation 4, PlayStation 3, Xbox One и Xbox 360. Игра выпущена компанией Electronic Arts 28 сентября 2018. Это игра-симулятор, дающая пользователю возможность испытать себя в роли тренера футбольного клуба - тренеровать, комплектовать команду и играть матчи.

         Дипломная работа направлена на такие аспекты игры, как комплектование и проведения трансферной политики команды. На основании параметров футболистов будет произведён поиск инсайтов, которые помогут пользователю игры определиться с трансферами, помогут выбрать футболиста с необходимыми качествами. Рассмотренная регрессионная модель даст понять, как будет развиваться игрок в следующих сезонах, что опять же улучшит понимание у пользователя игры о дальнейшей трансферной политике. Так же в работе будет предоставлена возможность выбрать лучших игроков из национальных чемпионатов.

         Источником данных является FIFA 19 complete player dataset

         Для обработки данных используется язык программирования Python. Для прогнозирования потенциального рейтинга футболиста взята модель линейной регресии. Для оценки качества модели используется среднеквадратичная ошибка.

            if int(command_input) <= len(country):
                i = 0

        На основании матрицы можно сделать выводы:
1. Параметры, которые используются для оценки голкиперов, мало коррелируются с параметрами полевых игроков, но проявляют сильную зависмость друг от друга
2. Потенциальный рейтинг слабо зависит от возраста, но сильно от текущего рейтинга и параметров игроков в поле
3. Многие параметры игроков не попали в матрицу - можно сделать вывод об отсутствии их влияния на основные параметры и в дальнейшем можно будет от них отказаться
         С помощью матрицы пользователь может определиться на какие другие параметры футболиста ему необходимо обратить внимание в первую очередь, если его интересует какой-либо конкретный параметр. Например, параметр "Marking" (опека) имеет сильную зависимость от "Aggression" (напор), "Interception" (перехват), "Standing" и "Sliding Tackle" (отбор стоя и в подкате), а это означает, что, при наличии хороших показателей в этих четырёх параметрах, игрок будет хорош в опеке, даже если ранее ему не ставили подобной задачи.

def simple_position(df):
    if (df['Position'] == 'GK'):
        return '0'
    elif ((df['Position'] == 'RB') | (df['Position'] == 'LB') | (df['Position'] == 'CB') | (df['Position'] == 'LCB') | (df['Position'] == 'RCB') | (df['Position'] == 'RWB') | (df['Position'] == 'LWB') ):
        return '1'
    elif ((df['Position'] == 'LDM') | (df['Position'] == 'CDM') | (df['Position'] == 'RDM')):
        return '2'
    elif ((df['Position'] == 'LM') | (df['Position'] == 'LCM') | (df['Position'] == 'CM') | (df['Position'] == 'RCM') | (df['Position'] == 'RM')):
