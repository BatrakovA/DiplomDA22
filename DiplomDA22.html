Введение

         Целью данной работы является анализ набора параметров футболистов игры FIFA 2019. Необходимо выявить информацию, которая поможет пользователю игры определиться с трансферами, выбрать игрока с необходимыми качествами и даст понять, как будет развиваться игрок в следующих сезонах.

         Источником данных является набор данных с сайта kaggle.com

         Для обработки данных используется язык программирования Python. Для прогнозирования потенциального рейтинга футболиста взята модель линейной регресии. Для оценки качества модели используется среднеквадратичная ошибка.

Анализ данных

         Данные содержат подробные параметры футболистов, присутствующих в последней версии базы данных FIFA 2019, такие как:

         Рассмотрим, какую информацию можно получить из указанных данных и как её может использовать пользователь игры.

def goat():

    Основная функция вывода лучших игроков в указанной стране
    """
    a = 0
    j = 0
    while j < len(country) : 
        print( str(j + 1) + '. ' + str(country[j]) )
        j += 1
    print('q - выход')
    while a < 1:
        command_input = input('Введите номер страны: \n')
        if command_input == 'q':
            break
        elif command_input.isdigit():
            if int(command_input) <= len(country):
                i = 0
                df_my = df_load[df_load['Nationality'] == country[int(command_input) - 1]]
                while i < len(pr_cols):
                    print('Лучший в {0} : {1}'.format(pr_cols[i],df_my.loc[df_my[pr_cols[i]].idxmax()][2]))

        На основании матрицы можно сделать выводы:

def simple_position(df):
    if (df['Position'] == 'GK'):
        return '0'
    elif ((df['Position'] == 'RB') | (df['Position'] == 'LB') | (df['Position'] == 'CB') | (df['Position'] == 'LCB') | (df['Position'] == 'RCB') | (df['Position'] == 'RWB') | (df['Position'] == 'LWB') ):
        return '1'
    elif ((df['Position'] == 'LDM') | (df['Position'] == 'CDM') | (df['Position'] == 'RDM')):
        return '2'
    elif ((df['Position'] == 'LM') | (df['Position'] == 'LCM') | (df['Position'] == 'CM') | (df['Position'] == 'RCM') | (df['Position'] == 'RM')):
        return '3'
    elif ((df['Position'] == 'LAM') | (df['Position'] == 'CAM') | (df['Position'] == 'RAM') | (df['Position'] == 'LW') | (df['Position'] == 'RW')):
        return '4'

        return '5'
    else:
        return df.Position
